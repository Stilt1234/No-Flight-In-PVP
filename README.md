# No Flight In PVP
### Disables a players Elytra for a minute when they get hit by another player (PVP).

# How the timer works :
When an elytra is equipped after getting hit by a player, it is removed and an action bar will tell you the time left before you can equip your elytra. Once the timer ends, you can use you Elytra freely. But, if you combat log i.e. logout during a fight, then the timer will be increased to 90 seconds as punishment.

# Features :
* Disable Elytra during PVP - Yes
* Prevent Elytra combat log - Yes
* Compatibility with 1.21.x - Yes
* Frustate runners - 100x Yes
