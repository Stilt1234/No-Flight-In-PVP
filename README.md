# No Flight In PVP
### Disables a players Elytra for a minute when they get hit by another player (PVP).

![No Flight In PVP](https://cdn.modrinth.com/data/cached_images/7b6d1a03c17161fd80b08042995e0e48e8f30495.webp)

# How the timer works :
When an elytra is equipped after getting hit by a player, it is removed and an action bar will tell you the time left before you can equip your elytra. Once the timer ends, you can use your Elytra freely. But, if you combat log i.e. logout during a fight, then the timer will be increased to 90 seconds as punishment.

# Features :
* Disable Elytra during PVP - ✅
* Prevent Elytra combat log - ✅
* Compatibility with 1.21.x - ✅
* Frustate runners - 100x ✅
